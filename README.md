
Project Structure
MainMenu.java: The central hub for navigating between different modules.

CustomerAccountWindow.java: Interface for creating and deleting customer records.

TransactionWindow.java: Interface for performing deposits and withdrawals.

HistoryWindow.java: Searchable table view for all transaction logs.

BankingOperations.java: Contains the core logic and SQL queries for database interaction.

DatabaseConnection.java: Manages the JDBC connection string and credentials.

Installation and Setup
Clone the repository.

Import the project into your IDE as a Maven or Gradle project.

Configure the database credentials in DatabaseConnection.java.

Run the SQL schema provided in the Database Configuration section.

Run MainMenu.java to start the application.
